# 卡通农场货仓助手

面向《卡通农场》中国大陆版本的网页库存规划工具，用于记录货仓/粮仓库存、设置目标库存、查看缺货优先级，并辅助判断生产链上的原材料需求。

## 功能

- 库存记录：维护当前数量、目标数量和隐藏状态。
- 缺货优先级：按目标差额和生产链需求辅助排序。
- 生产链查看：展示产品、原料、设备和生产时间。
- 数据编辑：可修改物品名称、分类、原材料、生产时间、图标等字段。
- 数据保存：通过内置服务保存 `data.json`，便于长期维护个人数据。

## 运行

```bash
npm install
npm start
```

然后打开：

```text
http://localhost:8766
```

也可以直接打开 `index.html` 预览页面；如果需要保存修改，请使用 `npm start` 启动服务。

## 校验

```bash
npm run test:data
npm run test:ui
```

第一次运行 UI 测试时，可能需要安装 Playwright 浏览器：

```bash
npx playwright install chromium
```

## 数据状态

- 当前数据覆盖 461 个物品、53 个分组。
- 生产数据以中文名称维护，目标是贴近中国大陆版本。
- 部分水果、鱼肉、礼券等非设备生产物会以 `0s` 或基础资源形式存在。
- 游戏版本、活动内容和中文译名可能变化，关键数据仍建议在使用中人工复核。

## 目录

- `index.html`：主应用。
- `data.json`：当前库存与生产数据。
- `icons/`：物品和设备图标。
- `server.js`：静态服务和保存接口。
- `validate.py`：数据一致性校验脚本。
- `wiki_products.json`：生产数据参考文件。
- `tests/`：浏览器冒烟测试。
